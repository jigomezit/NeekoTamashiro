# Imágenes Requeridas

Coloca las siguientes imágenes en esta carpeta (`/public/img/`) con los nombres exactos especificados:

## Lista de Imágenes

1. **hero-background.jpg**
   - Imagen de fondo del Hero Banner
   - Debe mostrar tatuajes japoneses en espaldas/brazos
   - Dimensiones recomendadas: 1440x1012px o mayor

2. **neeko-profile.jpg**
   - Foto de perfil de Neeko Tamashiro para la sección Biography
   - Dimensiones recomendadas: 406x475px o proporción similar

3. **book-1.jpg**
   - Primera imagen de Book Features
   - Dimensiones recomendadas: 296x296px o mayor (cuadrada)

4. **book-2.jpg**
   - Segunda imagen de Book Features
   - Dimensiones recomendadas: 296x296px o mayor (cuadrada)

5. **book-3.jpg**
   - Tercera imagen de Book Features
   - Dimensiones recomendadas: 296x296px o mayor (cuadrada)

6. **artwork-background.jpg**
   - Imagen de fondo de la sección Art Work
   - Debe mostrar la deidad roja con llamas (arte japonés tradicional)
   - Dimensiones recomendadas: 1440x960px o mayor

7. **kabuto-project.jpg**
   - Imagen del proyecto Kabuto Rotary Machines
   - Dimensiones recomendadas: 406x422px o proporción similar

8. **kamisama-project.jpg**
   - Imagen del proyecto Kamisama
   - Dimensiones recomendadas: 406x422px o proporción similar

9. **merch-background.jpg**
   - Imagen de fondo de la sección Merch
   - Debe mostrar stickers y patches con diseños de tatuajes japoneses
   - Dimensiones recomendadas: 1450x710px o mayor

10. **logo-icon.png** (opcional)
    - Logo circular del header
    - Puede ser PNG o SVG
    - Dimensiones recomendadas: 48x48px

## Notas

- Todas las imágenes serán optimizadas automáticamente por Next.js
- Los formatos soportados son: JPG, PNG, WebP
- Asegúrate de que las imágenes tengan buena calidad y resolución adecuada
